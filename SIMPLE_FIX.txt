COPY THESE COMMANDS ONE BY ONE TO YOUR RASPBERRY PI
====================================================

THE PROBLEM: Your venv is using Python 3.13, but Vosk doesn't work with 3.13
THE SOLUTION: Force Python 3.11 OR install system-wide

-----------------------------------
OPTION 1: Force Python 3.11 (BEST)
-----------------------------------

cd ~/respeaker-ai
deactivate
sudo apt-get install -y python3.11 python3.11-venv python3.11-dev
rm -rf venv
python3.11 -m venv venv
source venv/bin/activate
pip3 install --upgrade pip
pip3 install cffi sounddevice vosk google-generativeai pyttsx3
python3 hello_ai_rpi.py


-----------------------------------
OPTION 2: System-Wide Install (EASIEST)
-----------------------------------

cd ~/respeaker-ai
deactivate
rm -rf venv
sudo pip3 install --break-system-packages cffi
sudo pip3 install --break-system-packages sounddevice  
sudo pip3 install --break-system-packages vosk
sudo pip3 install --break-system-packages google-generativeai
sudo pip3 install --break-system-packages pyttsx3
python3 hello_ai_rpi.py


-----------------------------------
OPTION 3: Use Emergency Script
-----------------------------------

cd ~/respeaker-ai
git pull
chmod +x emergency_fix.sh
bash emergency_fix.sh
